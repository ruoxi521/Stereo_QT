# 使用介绍
- 相机这里输入相机索引
- 模式这里输入模式索引
1. 左单目模式：LEFT_EYE_MODE
2. 右单目模式：RIGHT_EYE_MODE
3. 红蓝模式：RED_BLUE_MODE
4. 双目模式：BINOCULAR_MODE
- 拍照
- 录像

# 运行方法

添加执行权限
```
sudo chmod 777 webcam_Stereo.sh
```

运行启动脚本
```
./webcam_Stereo.sh
```
